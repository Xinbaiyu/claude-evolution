### 2. history 命令 (9/9 任务完成)

**功能**: 显示审批历史记录

**实现文件**: `src/cli/commands/history.ts`

**特性**:
- ✅ 读取已批准和已拒绝的建议
- ✅ 按时间倒序排列（最新的在前）
- ✅ 支持 `--limit` 参数（默认 10 条，可自定义）
- ✅ 支持 `--type` 参数（all/approved/rejected 过滤）
- ✅ 表格格式输出（使用 cli-table3）
- ✅ 彩色显示（✓ 绿色批准，✗ 红色拒绝）
- ✅ 智能时间显示（刚刚/N 小时前/N 天前/日期）
- ✅ 描述自动截断（防止超长文本）
- ✅ 置信度颜色编码（高=绿色，中=黄色，低=灰色）

**使用示例**:
```bash
# 查看最近 10 条历史
claude-evolution history

# 查看最近 20 条历史
claude-evolution history --limit 20

# 只查看已批准的
claude-evolution history --type approved

# 只查看已拒绝的
claude-evolution history --type rejected
```

**输出示例**:

### 3. diff 命令 (10/10 任务完成)

**功能**: 显示原始配置与进化配置的差异

**实现文件**: `src/cli/commands/diff.ts`

**特性**:
- ✅ 读取原始 CLAUDE.md（source 目录）
- ✅ 读取进化后的配置文件（learned 目录）
- ✅ 使用 diff 库生成行级差异
- ✅ 彩色差异输出（+ 绿色=新增，- 红色=删除，灰色=未变更）
- ✅ 支持 `--no-color` 选项（纯文本输出）
- ✅ 差异统计摘要（新增/删除/未变更行数）
- ✅ 处理文件缺失场景（友好提示）
- ✅ 合并多个 learned 文件

**使用示例**:
```bash
# 查看配置差异（彩色）
claude-evolution diff

# 查看配置差异（纯文本，便于重定向）
claude-evolution diff --no-color > changes.txt
```
