**核心依赖** (9 个):
- `@anthropic-ai/sdk` - LLM API
- `@modelcontextprotocol/sdk` - MCP 协议
- `chalk` - 终端颜色
- `chokidar` - 文件监听
- `commander` - CLI 框架
- `fs-extra` - 文件操作
- `node-cron` - 定时任务
- `uuid` - ID 生成
- `zod` - Schema 验证

**开发依赖** (7 个):
- TypeScript 及类型定义
- `tsx` - TS 执行器
- `vitest` - 测试框架 (未使用)

### 数据流

```
┌─────────────────┐
│  claude-mem MCP │ 历史会话数据
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│  MCP Client     │ 三层检索 + 过滤
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│  LLM Extractor  │ 提取偏好/模式/工作流
└────────┬────────┘
         │
         ↓
┌─────────────────┐
│  Learner        │ 冲突检测 + 阶段判断
└────────┬────────┘
         │
         ├─→ 自动应用 → learned/
         └─→ 待审批   → suggestions/
                │
                ↓
         ┌─────────────────┐
         │  MD Generator   │ 拼接生成
         └────────┬────────┘
                  │
                  ↓
         ┌─────────────────┐
         │  CLAUDE.md      │ 最终配置
         └─────────────────┘
```
